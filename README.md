## Overview
A sleek and user-friendly music player built using Tkinter, designed for a professional and aesthetically pleasing UI. This project features a transparent song list box and label, ensuring a seamless and immersive experience.

## Features
- **Minimalistic UI**: Clean and elegant design with a background image.
- **Transparent Song List**: The playlist is displayed directly on top of the background without a black box.
- **Basic Music Controls**: Play, Pause, Resume, Stop, and Next functionality.
- **File Support**: Supports common audio formats like MP3 and WAV.
- **Playlist Management**: Load multiple songs and manage your playlist easily.

## Technologies Used
- **Python**
- **Tkinter (GUI Framework)**
- **Pygame (Audio Processing)**

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/pawniityagii/music-player.git
   ```
2. Navigate to the project directory:
   ```bash
   cd music-player
   ```
3. Install dependencies:
   ```bash
   pip install pygame
   ```
4. Run the application:
   ```bash
   python music_player.py
   ```

## Usage
1. Open the application.
2. Click on **Load Songs** to add music files from your system to the playlist.
3. Use the controls to play, pause, next, prev, or stop the music.
4. Enjoy your personalized music experience!
